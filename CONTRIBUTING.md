# Contributing to Awesome Pokémon Showdown

Thank you for helping maintain this curated list!

## Adding a Resource

1. Fork this repository
2. Add your suggestion to the relevant section in README.md
3. Submit a pull request with a brief description of why the resource belongs

## Inclusion Criteria

- Actively maintained (updated within 2 years, or permanently stable)
- Directly related to Pokémon Showdown usage or development
- Free or open source

## Format

Tables use the format:

```markdown
| [Name](URL) | Description | Additional column if needed |
```

## What's Not Included

- Abandoned tools (last commit 3+ years ago with known bugs)
- Closed-source paid tools without a meaningful free tier
- Resources primarily for formats outside Showdown (e.g., cartridge-only tools)
